SimpleHatebuViewer-iOS
========================

iPhone6 plusを購入したものの、広い解像度を活かせるはてなブックマークビューアがなかったので、自分でつくってみました。  
公開の予定はなかったのですが、くれくれリクエストをいただいたので公開します。  
ほんとたいしたものじゃないので、一応MITライセンスはつけてますが、再利用はご自由に。
